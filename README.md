# 🏥 Flutter Healthcare System

A multi-user Flutter healthcare application consisting of separate interfaces for patients and doctors.

This project demonstrates a healthcare ecosystem with modern Flutter UI, role-based application structure, and scalable frontend architecture.

---

## 📱 Modules

### 👤 Patient App
Features:
- Appointment booking
- Doctor browsing
- Health records access
- User authentication
- Responsive UI

### 🩺 Doctor App
Features:
- Patient management
- Appointment handling
- Dashboard interface
- Medical record updates
