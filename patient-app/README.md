# 👤 HealthConnect Patient App

A Flutter-based patient-side healthcare application designed to help users manage appointments, connect with doctors, and access healthcare services through a modern mobile interface.

This project is part of the **Flutter Healthcare System**, consisting of separate applications for patients and doctors.

---

## 🚀 Features

- Patient dashboard
- Doctor browsing
- Appointment booking
- Responsive Flutter UI
- Cross-platform support
- Clean and user-friendly interface

---

## 🛠️ Technologies Used

```bash

```

```bash
flutter pub get
```

### 3. Run Application

```bash
flutter run
```
